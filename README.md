## Descripción del proyecto

Medify es una plataforma enfocada en la gestión y consulta de medicamentos disponibles en farmacias, permitiendo que los usuarios puedan verificar desde una aplicación si un medicamento se encuentra disponible antes de desplazarse físicamente al establecimiento.

Además, el sistema incorpora funcionalidades relacionadas con la gestión de información médica, como historiales clínicos, recetas médicas, pacientes y doctores.

El proyecto fue desarrollado utilizando una base de datos relacional desplegada en la nube mediante Supabase y PostgreSQL.

---

## Problemática abordada

Muchas personas deben desplazarse hasta diferentes farmacias, realizar largas filas y esperar atención para finalmente descubrir que el medicamento que necesitan no se encuentra disponible.

Medify busca solucionar esta problemática permitiendo visualizar disponibilidad y stock de medicamentos desde una aplicación web.

## Consultas realizadas

- Historial clínico de pacientes.
- Valor económico del inventario.
